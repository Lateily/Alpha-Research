# 五轴评分量规 — Core Thesis 研究质量测量仪器

| | |
|---|---|
| 用途 | 给任意一份"单票投研材料"在五个维度上打分(0-100/轴),可复现、可对比 |
| 测量对象 | **写作质量(论点的具体性/可证伪性/估值纪律/信息增量/风险识别)**,**不是**论点最终对不对(那是前向 checkpoint 的事) |
| 谁打分 | **人工红队 = measure of record**(Junyan 的五轴是档案级裁决);**auto/盲打分 = floor only**(可复现的下限,用于基线对照与一致性交叉验证) |
| 来源 | CTF v1 spec §4 的五轴定义 + BYD/华海/鼎龙红队评语的实证锚 |

> 测量"写得空不空",不测"猜得对不对"。后者由 2026-07-10 起的 30/60/90 前向检查点回答。

## 五轴(每轴 0-100,band 划分)

### 轴1 — 具体性 Specificity(可执行的具体 vs 空泛叙事)
- **85-100**:明确事件/日期/数值阈值/价格区间/盈亏比/仓位档位,判断落到可核对的数字
- **70-84**:多数判断有具体数字,部分仍定性
- **50-69**:有些数字但主要是定性叙事("增长强劲""估值偏高")
- **<50**:几乎全空泛,无可执行具体点

### 轴2 — 可证伪性 Falsifiability(论点能被什么证伪,机制化吗)
- **85-100**:明确 wrong-if 触发器,每个带 **指标+阈值+来源+日期**,可机械判定;有明确证明/证伪事件
- **70-84**:有可证伪条件但部分未机制化(缺阈值/来源/日期)
- **50-69**:隐含可证伪但没写明触发条件
- **<50**:不可证伪("长期看好""基本面扎实"),没有"什么发生我就错了"

### 轴3 — 估值纪律 Valuation discipline(估值有锚有场景有纪律吗)
- **85-100**:多场景区间(熊/基/牛)各带显式假设+模型锚+盈亏比对照入场纪律;结论可追溯到假设
- **70-84**:有区间和假设但部分锚是 analyst-constructed,或缺盈亏比纪律
- **50-69**:给了 PE/估值水平判断但无场景无显式假设("80倍偏高")
- **<50**:估值只是一句标签("贵了""便宜")或缺失

### 轴4 — 信息增量 Information increment(相对市场共识/普通摘要有没有增量认知)
- **85-100**:明确 variant perception——市场信X我们信Y,且Y来自非共识的具体事实/拆分;指出共识在哪、自己边际在哪
- **70-84**:有差异化视角但部分是共识的精致复述
- **50-69**:主要是把卖方/共识观点整理复述,增量有限
- **<50**:纯共识复述,任何人从公开摘要都能得到同样结论

### 轴5 — 风险识别 Risk identification(风险全面、具体、可跟踪吗)
- **85-100**:覆盖基本面/政策/估值/流动性/拥挤度各类,每类具体且可量化/可跟踪,承认数据缺口
- **70-84**:覆盖多数类别,多数具体
- **50-69**:列了主要风险但偏笼统("估值高""周期性")
- **<50**:风险一笔带过或缺失

## 打分纪律(防自欺)
1. **不奖励格式/长度**:一份散文如果同样可证伪、估值同样有纪律,拿同样高分;一份决策书如果只是堆表格而无机制化 wrong-if,不因格式得分。
2. **逐轴引证据**:每个分必须引材料里的具体内容作为依据,不准黑箱给分。
3. **盲打分用于对照**:做基线对照时,材料匿名化(剥 provenance)、随机标签、打分者不知道哪份是"我们的"——这是 STRATEGY_LAB_AGENT.md 角色隔离原则在研究侧的落地。
4. **bar(参考,[unvalidated]):平均 ≥70 且无单轴 <50** 才算合格决策书(CTF spec §4);auto 盲打分给的是 floor,人工红队给的是档案。

bar 阈值与轴定义本身是 `[unvalidated intuition]`,会随更多样本和 Junyan 的红队记录校准。

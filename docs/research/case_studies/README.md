**Author:** Junyan (Myers) Liu | LSE BSc Economics
**Context:** UBS Finance Challenge 2026 — "Chinese Companies Going Global"
**Position:** Long 300308.SZ Innolight / Short 300394.SZ T&S Communications
**Last update:** April 2026 (v5 Final, post-Q1 2026 earnings)

**Structural strengths to learn from:**
- **6-pillar framework** (L1/L2/L3 long + S1/S2/S3 short) — each pillar
  is "what market sees vs what we see independently"
- **Audit-grade evidence** — every claim backed by annual report page
  numbers + verifiable financial data, not narrative
- **Contrarian risk treated, not avoided** — e.g., "inventory +105% looks
  like risk; here are the 3 evidence chains showing it's healthy build"
- **Wrong-if conditions per pillar** — every L/S has explicit kill switch
- **Profit pool migration thesis** — both stocks in same secular tailwind,
  but profit asymmetry within value chain (rare structural insight)

## Anti-pattern guard

PM in an interview." If a draft isn't there yet, don't put it here.

For drafts and exploratory work, use a different location (or keep on local
disk until ready).

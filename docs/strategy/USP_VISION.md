## 核心 USP：中国权益研究的双层认知系统

### 为什么现有工具做不到

全球 99% 的中国股票研究工具只能做到其中一种：

**国内视角（卖方研究员 / A 股 PM）：**
- ✅ 懂政策信号和文化语境
- ❌ 分析框架不够严谨，国际估值体系薄弱
- ❌ 对跨市场比较和国际定价逻辑不熟悉

**国际视角（外资机构 / 海外学者）：**
- ✅ 框架严谨（DCF / comparable / factor model）
- ❌ 看不透政策逻辑，政策解读依赖翻译过滤后的信息
- ❌ 对国内投资者叙事和市场情绪缺乏感知

**没有任何工具同时系统性地做到两者。**

这个 USP 锚定在团队身份本身：
- Junyan（LSE）：国际投资框架训练 + 母语中文 + 国内市场直觉
- Franky（MIT）：顶尖量化训练 + 外部视角 + 机构思维
- AI Stack：把上面两种视角都系统化、持续化

## 三层护城河

### 层 1：政策信号解码

**What it is:**
把 CSRC 发文、五年规划细则、产业政策、地方补贴——翻译成具体公司的影响链条。

**Why it matters:**
99% 外资机构 PM 不具备这个能力。
国内卖方有政策敏感度但缺国际框架严谨度。

**具体实现（待建）：**
- 政策关键词追踪（CSRC 官网 + 国家发改委 + 工信部）
- 行业受益传导图谱（政策 → 受益行业 → 受益公司）
- 历史政策命中率追踪（我们判断政策影响的准确率）

**不可复制的原因：**
需要从头积累政策解读 + 公司影响验证的历史数据
机器可以做信号提取，但框架本身需要人类判断校准

---

### 层 2：国内叙事 Gap 分析

**What it is:**
追踪国内散户/投资者在雪球/东方财富评论区在炒的叙事 vs 国际机构的定价逻辑
→ Gap 本身就是 alpha：国内过度悲观 = 潜在低估；国内过度乐观 = 潜在高估

**Why it matters:**
大多数量化模型不抓社交叙事
大多数定性研究不量化叙事 gap
两者都做的，目前没有工具

**具体实现（待建）：**
- 雪球 / 东财评论爬取（情绪标注）
- 叙事主题识别（主要在聊什么）
- Gap 分数计算（国内 vs 国际 sentiment 差距）
- 历史 Gap 与后续超额收益的相关性校验

**不可复制的原因：**
需要建立中文社交媒体爬取 + 情绪标注体系
需要长时间积累 gap → 收益数据才能验证

### 层 3：跨框架收敛层

**What it is:**
同一只股票，用 6-10 个投资大师框架同时分析：
- 国际派：Buffett（护城河/ROE），Graham（安全边际），Damodaran（估值严谨），Burry（逆向/债务）
- 中国派：段永平（产品/商业模式），邓晓峰（ROE/复利），张磊（长期价值）
- 宏观派：Dalio（债务周期），Howard Marks（周期位置）

找多框架交集 = 高确定性信号；框架分歧 = 需要深挖的不确定性

**Why it matters:**
国际 + 国内框架同时跑，目前没有任何工具做到
已有 Buffett/Burry/Damodaran 三个 persona（v15 已部署），需要扩展到 8-10 个

**具体实现（已有雏形，待扩展）：**
- `scripts/persona_overlay.py` → 扩展到 8+ personas
- 增加国内派系（段永平、邓晓峰的投资思想体系化）
- 框架交集 / 分歧可视化

## USP 的第四层（框架本身就是 USP）

你在对话中提出的重要认知：**USP 不是一个孤立的卖点，是整个闭环研究体系的成熟度本身。**

> 把所有策略角度分类写到一个文件中，每次进行筛选分析股票时会过一遍，
> 你永远不会漏掉任何一点。

这就是 `INVESTMENT_FRAMEWORK.md` 的用途 — 40+ 投资视角库，覆盖所有可能遗漏的分析维度。

**当一个分析师的工具永远不会漏掉任何一个维度、永远不受情绪影响、永远同时持有六个互相矛盾的投资哲学——这个分析师本身就是 USP。**

## 商业化路径（预留，暂不是重点）

当框架足够成熟后（推测 6-12 个月后），最自然的商业化形式：

1. **买方辅助工具（SaaS）：** 给中小型本土基金的研究员，帮他们覆盖他们平时顾不上的港股/海外中概
2. **研究报告订阅：** 每周生成 1-2 份 "经得起Franky挑刺" 的完整研报
3. **策略咨询：** 帮有意进入中国市场的境外机构进行政策信号解读

**但先决条件是框架真正可靠：**
- Prediction hit rate ≥ 65% at n ≥ 20（目前 67% at n = 3，样本太小）
- wrongIf 触发的止损确实保护了收益
- 能展示 bridge 1a-2-3 都已连通的完整 pipeline

（但维护模块如果是修复破损的东西，优先级提升。）
